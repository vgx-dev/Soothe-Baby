# Soothe Baby - Brand Identity Guide

## Overview
Soothe Baby is a premium sensory touch app for infants (0-2 years old) designed for Product Hunt launch. The visual identity communicates "calm joy" - premium quality that feels safe for parents while remaining playful and delightful for babies.

## Brand Colors

### Primary Brand Anchor
- **Brand Blue**: `#1A73E8` - Used consistently across all screens, icons, and marketing materials

### Supporting Palette
- **Purple**: `#7B2CBF`
- **Pink**: `#FF006E`
- **Yellow**: `#FFD60A`
- **Mint**: `#06FFA5`
- **Orange**: `#FCA311`

## Typography

### Headings
- **Font**: Baloo 2 (Google Fonts)
- **Weights**: 600-800
- **Usage**: App name, feature titles, CTAs, all headings
- **Character**: Rounded, confident, friendly

### Body Text
- **Font**: Fredoka (Google Fonts)
- **Weights**: 400-600
- **Usage**: Descriptions, body copy, UI labels
- **Character**: Approachable, easy to read

## Design Principles

### 1. Glassmorphism
- Soft backdrop blur effects (backdrop-blur-xl, backdrop-blur-2xl)
- Semi-transparent backgrounds (white/10 to white/25)
- Subtle borders (border-white/20 to border-white/40)
- Layered depth with shadows

### 2. Rich Gradients
- Multi-stop gradients for backgrounds
- Gradient overlays on shapes and UI elements
- Animated gradient orbs for ambient motion
- Brand blue always present in main gradients

### 3. Strong Color Saturation
- Vibrant, eye-catching colors
- High contrast against backgrounds
- Glowing effects with matching color shadows
- Safe color palette for infant visual development

### 4. Smooth Animations
- Motion library for fluid animations
- Floating shapes with easing
- Hover states on interactive elements
- Welcome animations and transitions

## App Icon

### Design
- Rounded square (28% border radius)
- Gradient from Brand Blue (#1A73E8) → Purple (#7B2CBF) → Pink (#FF006E)
- Centered sparkle icon in white
- Subtle glow overlay at top-left (30% opacity)
- Shadow for depth

### Sizes Available
- 120px (primary)
- 80px (medium)
- 60px (small)
- 40px (icon)

## Interactive Elements

### Shapes
1. **Circle**: Solid gradient circles with matching glow
2. **Star**: SVG stars with blue gradient
3. **Heart**: SVG hearts with pink gradient
4. **Bubble**: Translucent circles with highlight accents

### Animations
- Pop-in scale animation (0 → 1.2 → 1)
- Float upward with random drift
- Rotation during float
- Fade out at end
- 4-second total duration

## UI Components

### Buttons
- **Primary**: Blue gradient with shadow
- **Secondary**: White glassmorphic with blur
- **Minimal**: Subtle white background
- All buttons: rounded-full, scale on hover/tap

### Glass Cards
- Background: white/15 with backdrop-blur-xl
- Border: white/30
- Padding: 6 (24px)
- Border radius: 3xl (24px)

### Controls
- Circular glassmorphic buttons (56px diameter)
- Icon size: 24px
- White icons with 2.5 stroke width
- Scale animations on interaction

## Screens

### Landing Page
- Full-screen gradient background
- Animated gradient orbs
- Centered app icon with animation
- Feature grid (2 columns on desktop)
- Large CTA button
- Age indicator footer

### Interactive Canvas
- Same gradient background as landing
- Tap/touch to spawn shapes
- Glassmorphic controls (home, sound)
- Bottom instruction pill
- Welcome message (auto-dismiss after 3s)

### Product Hunt Preview
- White background showcase
- Gallery thumbnail mockup
- Icon size variations
- Color palette swatches
- Typography samples
- Design principles overview

## Emotional Tone

**Calm Joy** = Premium + Playful + Safe

- **Premium**: Polished animations, glassmorphism, sophisticated gradients
- **Playful**: Vibrant colors, bouncy typography, delightful interactions
- **Safe**: Rounded corners everywhere, soft shadows, gentle animations

## Product Hunt Launch Checklist

✅ Consistent brand blue (#1A73E8) across all assets
✅ App icon optimized for gallery thumbnails
✅ Landing page demonstrates premium quality
✅ Interactive demo showcases core value
✅ Typography creates confident, friendly impression
✅ Color palette is vibrant and memorable
✅ All animations are smooth and delightful
